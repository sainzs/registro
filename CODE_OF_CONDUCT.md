# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Pledge

We pledge to make participation in this project respectful, inclusive, and
focused on useful agent work reporting. Contributors should be able to ask
questions, report issues, suggest changes, and review work without harassment,
insults, or personal attacks.

## Expected Behavior

- Be kind, direct, and specific.
- Assume good intent while still asking for evidence.
- Keep reviews focused on the work, not the person.
- Respect boundaries around private data, credentials, employer code, and user
  runtime state.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private information or credentials.
- Repeatedly pushing broad, unsafe, or off-topic changes after maintainers ask
  you to stop.

## Enforcement

Maintainers may edit, hide, reject, or remove contributions that violate this
code. Serious or repeated violations may lead to blocked participation in this
project.

Report concerns privately to the repository owner through GitHub.
